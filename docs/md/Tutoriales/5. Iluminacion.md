
# 1. Ambient
Es la luz ambiental que tienen de por si los objetos, simplemente es reducir la luz que se emite.
```cpp
vec3 ambient = ambientStrength * lightColor;
```

# 2. Diffuse
Es la luz que reciben los objetos desde el foco de luz. Como de iluminado esta la zona se define por el ángulo entre el rayo de luz y la normal en ese punto.  Se hace el producto escalar de ambos para saber como están, si es 0 es que están a 90º grados y no recibe luz
```cpp
 vec3 norm = normalize(Normal);

 vec3 lightDir = normalize(lightPos - FragPos);

 float diff = max(dot(norm, lightDir), 0.0);

 vec3 diffuse = diff * lightColor;
```

# 3. Specular
Es la luz ambiental que refleja el objeto basado en las propiedades del material. Se calcula en función de como estén el vector que se refleja de la superficie y el vector que va desde donde miras al punto de impacto del rayo de luz.
```cpp
float specularStrength = 0.5;

vec3 viewDir = normalize(cameraPos - FragPos);

vec3 reflectDir = reflect(-lightDir, norm);

float spec = pow(max(dot(viewDir, reflectDir), 0.0), 16);

vec3 specular = specularStrength * spec * lightColor;
```


# 4. Materiales
- Ambient: el color de la superficie
- Diffuse: el color bajo la iluminación difusa
- Specular: color de la luz de los reflejos
- Shininess: el radio de la luz specular.
